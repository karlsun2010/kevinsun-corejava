《操作系统原理》

高并发下，并发编程是必备技能。

并发编程模型： 管程和信号量

并发编程三个核心问题：
1 分工：如何高效地拆解任务并分配给线程。
2 同步：线程间如何协作。
3 互斥：同一时刻只允许一个线程访问共享变量。

java sdk并发包只是针对并发问题开发出来的工具而已。


并发编程并不是java特有的语言特性，java只是根据自身情况作了实现而已。


学习攻略：
1 分工
2 同步： 线程之间依赖关系，一个线程执行完任务，如何通知执行后续任务的线程开工。
3 互斥： 线程安全问题，可见性，原子性，有序性。

实现互斥的核心技术就是锁。Synchronized ，Lock
锁解决安全性问题的同时也带来了性能问题。
保证安全，并尽可能提高性能。分场景优化。
